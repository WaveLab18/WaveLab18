What changed

변경 포인트

This version removes the extra channel-guide image section and turns the active media area into the main interaction zone.

채널 안내 이미지를 제거하고, 운영 중인 미디어 영역 자체가 핵심 인터랙션이 되도록 구조를 바꿨습니다.

Less clutter

덜 복잡한 구조

The guide image section is removed so the site feels tighter and more intentional.

채널 안내 이미지 섹션을 빼고 핵심 콘텐츠 중심으로 정리했습니다.

Better channel identity

채널별 정체성 강화

Each channel now has its own representative image card and its own preview flow.

각 채널마다 대표 이미지를 따로 두고, 채널별 미리보기 흐름을 분리했습니다.

Clickable actions

클릭 기능 강화

Visitors can open the studio gallery, preview a YouTube video, and jump directly to the channel.

스튜디오 갤러리, 유튜브 미리보기, 채널 바로가기 기능을 넣었습니다.

Studio section

스튜디오 섹션

Use the button to open your two studio images in a modal gallery.

The first screen stays clean, while the gallery still feels interactive.
